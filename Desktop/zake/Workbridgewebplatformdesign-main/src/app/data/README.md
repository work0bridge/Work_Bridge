# Data Structure

This folder contains static and demo-oriented data used by the frontend.

## Files

- `jobsData.ts`: job listing seed data and job-related display content
- `platformData.ts`: broader platform seed data for dashboards, profiles, services, finance, and other demo screens
- `index.ts`: grouped exports for easier imports

## Purpose

These files support the current frontend experience with ready-to-use local data and help keep page files lighter.
