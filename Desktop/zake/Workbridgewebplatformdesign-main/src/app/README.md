# App Structure

This folder is organized into clear sections so the project stays easy to navigate:

- `components`
- `data`
- `pages`
- `route-config`
- `storage`

## Components

- `components/layout`: shared page layouts
- `components/shared`: reusable feature sections
- `components/ui`: low-level UI primitives

## Pages

- `pages/public`: landing and public-facing pages
- `pages/auth`: login, register, and account access flows
- `pages/user`: user dashboard and user features
- `pages/company`: company dashboard and company features
- `pages/admin`: admin dashboard and admin features

## Data And State

- `data`: static seed/demo data
- `storage`: local storage helpers and app-side state persistence

## Routing

- `routes.ts`: final router creation
- `route-config`: grouped route definitions by app area
